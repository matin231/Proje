این پروژه یک سیستم فروشگاهی (Shop Management System) با زبان Java و رابط گرافیکی Swing است که به‌صورت مرحله‌ای توسعه داده شده و امکان مدیریت کاربران، محصولات، سبد خرید و فرآیند خرید نهایی را فراهم می‌کند.

هدف پروژه، پیاده‌سازی مفاهیم برنامه‌نویسی شی‌گرا (OOP)، کار با فایل (CSV)، و طراحی یک نرم‌افزار فروشگاهی قابل اجرا برای درس دانشگاهی می‌باشد.

امکانات اصلی
 ورود و ثبت‌نام کاربران (Customer / Admin)
 مدیریت نقش کاربران
 مدیریت محصولات (افزودن، حذف، جستجو، مرتب‌سازی)
 سبد خرید برای مشتریان
 بررسی موجودی و انجام خرید نهایی
 ذخیره اطلاعات در فایل‌های CSV
 رابط گرافیکی با Java Swing

Phase 2: سیستم فروشگاه و سبد خرید
Phase 3: پنل ادمین و خرید نهایی
